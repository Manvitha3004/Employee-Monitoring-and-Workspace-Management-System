📋 Camera Not Working? Troubleshooting Tips:

  • Access via correct URL: Use http://127.0.0.1:5000 (NOT localhost)
  • Allow permissions: Click "Allow" when browser asks for camera access
  • Check browser settings: Ensure camera is enabled in browser settings
  • Close other apps: Exit Zoom, Teams, Skype, or other apps using the camera
  • Windows Privacy: Settings → Privacy → Camera → Allow desktop apps to access camera
  • Try different browser: Chrome or Edge work best for camera access
  • Check camera connection: Ensure USB camera is properly connected

Still having issues? Check the browser console (F12) for detailed error messages.
